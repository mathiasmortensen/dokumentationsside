Why Node.js?

Node.js makes it possible to use JS without a browser, which i find very intriguing.

Not having to change language when dealing with both backend and frontend is nothing short of genius.

NPM the package manager is integrated into Node.js. This means thousands of libraries, at your disposal.

Node.js is a fast and reliable way to get started with boilerplate free MVP applications.

Go to nodejs.org
