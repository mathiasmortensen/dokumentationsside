Introduction to JS

Variables

Always use const by default unless the value needs to change.

const is constant in the assignment. Example:


const myNumber = 4;

console.log(myNumber);

// This prints 4

Common Data Types:

String, Boolean, Number, BigInt, Object, Symbol, null, undefined


const person = {

    // key-value pair

    name: "Mathias"

};

person.name = "Gustav";

person.age = 10021132134;

console.log(person);


const people = [];

people.push("Victor");

console.log(people);

How to define strings:

single quotes ' ', double quotes " ", and backticks ` `

Single quotes offer no string interpolation (variables cannot be inserted directly). Escape sequences are also required when trying to use the same quote inside the string. Example:


const textExample = 'My name is Mathias\' and I like apples.';

Double quotes " "

Double quotes offer the exact same functionality as single quotes, the only difference is the character.

At last we have the infamous backticks... ` `

Backticks can be used for string interpolation and for multiline strings.


let importantChangingNumber = 17;

const needsImportantNumber =
`The Super Important Changing Number For Today: ${importantChangingNumber}`;

console.log(needsImportantNumber);

// The Super Important Changing Number For Today: 17

This will make console.log print:

The Super Important Changing Number For Today: 17
